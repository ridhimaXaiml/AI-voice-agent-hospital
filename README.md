# 🎙️ Voice Assistant Project
A Python-based Voice Assistant that listens to user commands, processes them, and performs automated tasks using speech recognition and text-to-speech.
## 🚀 Features
- Voice command recognition
- Text-to-speech response
- Basic automation tasks
- Database integration
- ## 🛠️ Tech Stack
- Python
- FastAPI
- Vapi
- SQLAlchemy
- REST APIs
- ## ⚙️ Run Project

git clone https://github.com/ridhimaXaiml/AI-voice-agent-hospital.git
pip install -r requirements.txt
python main.py
